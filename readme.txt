== Description ==

Adds  a simple sticky comment settings page and button.

<strong>This plugin is no longer under development</strong>

Their will be a settings page called Sticky Comments under the comments tab in the admin page.

To make a comment stciky, just edit the comment and select the sticky comment box.

Video Tutorial

[youtube https://www.youtube.com/watch?v=J0-Fyw98NUc]

= Please note!!! =

This plugin only works for users with dashboard access.

== Installation ==

1. Go to the Plugins> Add New> Upload in you admin dashboard.
2. Upload the zip file and install.
3. Go to the Plugins section in your admin dashboard and activate.
